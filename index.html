Liste med avendte Git commandoer

  • Git clone - cloner dit remote repoi til din computer
  • Husk at efter du har clonet skal du skrive touch og navnet.html/css/js
  • Git add -A - adder alle filer og ændringer til stage
  • git commit -m "Skal være relevent tekst"
  • Git push - skubber dit lokale repo ud i skyen
  • git status - tjekker status på dit LOKALE repo
